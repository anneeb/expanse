# Expanse

Expanse is a mapping tool that allows for visualization of parent and child relationships in three dimensional space.

## Instructions

In expanse/server:

* ```rake db:create```
* ```rake db:migrate```
* ```rails start```

In expanse/client:

* Open ```index.html``` in browser

## Technologies

Server:
* Ruby on Rails
* PostgreSQL

Client:
* Javascript
* jQuery
* Three.js
* ThreeX
* Materialize
